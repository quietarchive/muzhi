# 对镜 · 墓志史料对读

以上官婉儿墓志为试点：在拓片上点击文句，对照弹出**墓志原文**与**传世史料**。

## 本地预览

```bash
cd muzhi
python3 -m http.server 8080
```

打开 http://localhost:8080

## 数据

- 对读表：`data/shangguan-duidu.json`（由 Excel「差异清单」导出）
- 前端内嵌：`js/data.js`
- 拓片：`assets/shangguan-muzhi.png`

重新从 Excel 导出时，可在仓库内用 `.venv` 跑现有解析脚本，再生成 `js/data.js`。

## 交互说明

1. 左侧目录按生平阶段列出 D01–D15
2. 拓片上半透明色块为可点击句段（位置为初估，可再校准）
3. 点击后上下两层弹窗：上为墓志原文，下为传世史料卡片

## 差异类型

一致 · 增补 · 事实抵牾 · 立场差异 · 缺载

本页只呈现差异与原文，不判断史料真伪。
